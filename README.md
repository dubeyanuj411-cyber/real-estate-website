# Real Estate Website - Frontend Only

A modern, responsive real estate property listing website built with React, Vite, and Tailwind CSS.

## Features

- 🏠 Property Listings with Filtering
- 🔍 Search by City/Keyword
- ❤️ Favorites/Wishlist (localStorage)
- 🌓 Dark/Light Mode Toggle
- 📱 Fully Responsive Design
- 🖼️ Image Gallery Slider
- 🎨 Modern UI with Tailwind CSS

## Installation

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM
- LocalStorage for data persistence

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/          # Page components
├── data/           # Static/dummy data
├── assets/         # Images, icons
├── layout/         # Layout components
├── hooks/          # Custom React hooks
└── utils/          # Helper functions
```
